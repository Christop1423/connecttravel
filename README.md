# ConnectTravel - B2B Travel Marketplace

A modern **B2B platform** connecting Tour Operators (Suppliers) with Travel Agencies and Corporate Planners worldwide.

## ✨ Features
- Professional **Agent Dashboard** with KPI cards and booking pipeline
- **Supplier Portal** for inventory and performance tracking
- Clean, modern dark theme design
- Built with Next.js 15, TypeScript & Tailwind CSS
